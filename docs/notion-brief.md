# 과로사 방지 구현 브리프

## 핵심 방향

- 장르: 직장인 생존 시뮬레이션 + 미니게임.
- 목표: 09:00부터 18:00까지 스트레스 100, 체력 0을 피하면서 업무량을 0으로 만들기.
- 구조: 미니게임 5회 고정, 메인화면 멀티태스킹, 점심 이벤트, 상사/동료 이벤트.
- 개발 우선순위: 와이어프레임과 플레이 흐름 검증 후 비주얼 스타일 적용.

## MVP 범위

- 타이틀: 이름, 성별, 커피파/담배파 선택.
- 온보딩: 신입사원 안내서 형태 튜토리얼.
- 출근: 상사 유형 힌트 공개.
- 메인화면: HUD, 채팅, 전화, 아이템 슬롯, 이벤트 팝업.
- 미니게임: 이메일 분류는 실제 플레이, 회의 준비/오탈자 찾기는 판정 버튼으로 흐름 검증.
- 엔딩: 칼퇴 성공, 야근, 당일퇴사, 과로사.

## 다음 구현 후보

- 회의 준비 드래그 정렬.
- 보고서 오탈자 클릭 판정.
- 채팅별 실제 카운트다운과 자동 소멸.
- 상사 유형별 이벤트 확률 적용.
- 상태이상별 커서 흔들림/느려짐 효과.
