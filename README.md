# 과로사 방지

웹 브라우저에서 실행하는 직장인 생존 시뮬레이션 프로토타입입니다.

## 실행

정적 서버로 실행하세요.

```bash
python -m http.server 8791 --bind 127.0.0.1
```

브라우저에서 `http://127.0.0.1:8791/` 접속.

## 기술

- HTML
- CSS
- Vanilla JavaScript ES Modules

## 현재 구현

- Galmuri 픽셀 UI 스타일 시작화면
- 온보딩 화면
- 출근 연출
- 메인 업무 화면
- 이메일 분류 미니게임 기본 루프
- 엔딩 화면
