# 엔딩 일러스트 (assets/endings/)

엔딩 화면 좌측의 연출 일러스트. `src/scenes/ending.js`가 `assets/endings/{key}.svg`를
`<img>`로 불러온다. **파일이 없으면** 설명 텍스트 placeholder가 표시되고, 파일을 넣으면
자동으로 일러스트가 뜬다 (코드 수정 불필요 — img onload/onerror 처리).

## 사양 (증명사진과 동일한 방식)

- 형식: 독립 실행 가능한 **SVG** (PNG도 가능 — `<img>`로 동일하게 로드)
- 구도: **가로형 일러스트**. 표시 영역은 `object-fit: cover`라 정확한 비율은 자유 (살짝 크롭됨). 권장 viewBox 예: `0 0 320 188` (≈16:9~5:3 가로)
- 스타일: 픽셀아트 레트로 (게임 톤에 맞춤)
- 팔레트(참고): 외곽선 `#1d1f2e` · 노랑 `#ffd23f` · 빨강 `#ff4d4d` · 초록 `#3fc24a` · 파랑 `#3d8bff`
- 톤: 칼퇴=밝고 여유 / 야근=어둑한 사무실 / 당일퇴사=결연 / 과로사=흑백·무거움

> 파일을 이 폴더에 넣기만 하면 엔딩 화면에 자동 반영됩니다.
